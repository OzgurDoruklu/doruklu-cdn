# Doruklu CDN

This repository serves as the central Content Delivery Network (CDN) for all Doruklu sub-projects (main, toprak, ozgur, nurcan).
It hosts shared resources such as:
- CSS stylesheets (`style.css`)
- Reusable UI scripts (`ui.js`)
- Supabase Configuration (`supabase-config.js`)

Served via GitHub Pages at: `https://cdn.doruklu.com/`
